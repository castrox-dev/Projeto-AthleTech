## Estrutura do Projeto

O projeto é composto por duas partes principais:

- `Projeto_academia/academia_backend/`: Contém o backend da aplicação, desenvolvido com Django REST Framework.
- `Projeto_academia/academia_backend/static/`: Contém os arquivos estáticos do frontend (HTML, CSS, JavaScript).

## Funcionalidades

O protótipo inclui as seguintes funcionalidades:

- **Autenticação de Usuários:** Cadastro, login e verificação de token.
- **Gerenciamento de Usuários:** API para operações CRUD de usuários (restrito a administradores).
- **Planos:** API para listar e gerenciar planos da academia.
- **Matrículas:** Sistema para usuários se matricularem em planos.
- **Frontend Simples:** Páginas de Home, Login e Cadastro para demonstrar a integração.
